# hexo-split-tags

Separate tags with commas.

## Install「Test Mode」

```bash
git clone git@github.com:wdssmq/hexo-split-tags.git

cd hexo-split-tags
npm link

# Hexo Site
cd ~/wwwroot/blog.test
npm link hexo-split-tags
```

Add the following to the `blog.test/package.json` file.

```json
"hexo-split-tags": "^1.0.0"
```
